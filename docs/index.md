## Instalação

### Como instalar
> Rode o comando abaixo em seu terminal ou CMD!

    pip install authwa

### Como executar
> Importe o modulo e adicione a uma variavel, chamando a classe desejada!

    import authWA

    whats = authWA.WA()

    whats.enviar_msg()
